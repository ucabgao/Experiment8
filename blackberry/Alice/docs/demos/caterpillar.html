AliceJS Demos

AliceJS - (A Lightweight Independent CSS Engine) is a micro JavaScript library focused on using hardware-accelerated capabilities (in particular CSS3 features) in modern browsers for generating high-quality, high-end visual effects.

Caterpillar Demos

Book - The "book" effect generates book with the content provided in the child divs of the parent div, allowing the binding to be placed on any side of the parent div.


$a("book").book({
    pageClassName: 'page',      // class title of pages
    bookWidth: 600,             // integer relating to pixels
    bookHeight: 400,            // integer relating to pixels
    speed: '500ms',             // select a speed
    paging: 'single',           // single || double
    binding: "left",            // left || right || top || bottom || center || middle
    circular: true              // true || false
});
